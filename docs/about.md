# PapiAI

The best standalone AI agent library in PHP.

Framework-agnostic, minimal dependencies, type-safe. Build powerful AI agents that call tools, stream responses, and integrate with any PHP application.

## Features

- **Multi-Provider** — Anthropic, OpenAI, Google Gemini, Ollama, Mistral, Groq, Grok, DeepSeek, Cohere, Azure OpenAI — one API for all.
- **Tool Calling** — Define tools as functions or class methods with PHP attributes.
- **Structured Output** — Zod-like schema validation for LLM responses.
- **Streaming** — First-class streaming with text chunks and rich events.
- **Middleware Pipeline** — Logging, caching, retry, rate limiting.
- **Framework Bridges** — Official Laravel and Symfony integrations.
